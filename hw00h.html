Git

mkdir projects creates a directory called "projects" inside your current directory.

cd projects changes your directory to "projects".

git clone https://phidip@bitbucket.org/phidip/de10.git makes a local copy of the repository placed on this URL.

git remote remove origin removes the reference to the remote repo.

git remote add origin "your repo" creates a new connection to a remote repo.
